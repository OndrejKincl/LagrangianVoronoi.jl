## LagrangainVoronoi.jl

A numerical library for Lagrangian Voronoi method in Julia. 

# Install prerequisites

Run the following command in Julia terminal:
```
using Pkg
deps = [
  "Polyester",
  "WriteVTK",
  "CSV",
  "StaticArrays",
  "SparseArrays",
  "Match",
  "Parameters",
  "DataFrames",
  "Plots",
  "LaTeXStrings",
  "Krylov",
  "Measures",
  "SmoothedParticles"
]
Pkg.add(deps)
```
